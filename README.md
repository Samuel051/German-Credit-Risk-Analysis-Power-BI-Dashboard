# German Credit Risk Analysis & Power BI Dashboard

## Project Overview

This project analyzes borrower behavior, loan exposure, and credit risk patterns using the German Credit Risk dataset.

The objective was to simulate a real-world banking/financial analytics workflow by combining:

* SQL analytics
* Credit risk analysis
* Portfolio exposure analysis
* Power BI dashboarding

---

# Tools Used

* PostgreSQL
* SQL
* Power BI
* Excel

---

# Project Workflow

## 1. Data Cleaning

Performed:

* Data validation
* Data type checks
* Duplicate handling concepts using staging tables and ROW_NUMBER()
* Category standardization

---

## 2. SQL Analysis

Conducted portfolio and borrower analysis using:

* CASE statements
* CTEs
* Aggregations
* Views

# Key Insights

* Car loans dominated portfolio exposure
* Medium-term loans contributed the highest risk exposure
* Highly skilled borrowers received the largest average loans
* Vacation loans had the highest average credit amount
* Housing ownership represented the largest portfolio concentration
